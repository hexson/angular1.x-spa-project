### 全局API

#### # 内置的全局API

全局API用于执行常见任务的JavaScript函数集合

```html
<!-- 基础 -->
angular.bootstrap(element, [modules], [config]) ------------ 手动启动模块
angular.element(element) ----------------------------------- 返回一个element
angular.module(appName, [modules]) ------------------------- 注册或检索模块
angular.version -------------------------------------------- AngularJs版本信息


<!-- 转换 -->
angular.copy(source, destination) -------------------------- 数组或对象深度拷贝
angular.equals(obj1, obj2) --------------------------------- 判断两个对象是否相等
angular.extend(dst, src) ----------------------------------- 合并对象
angular.forEach(object, function) -------------------------- 对象或数组的迭代函数
angular.lowercase ------------------------------------------ 字符串转换为小写
angular.uppercase ------------------------------------------ 字符串转换为大写


<!-- 比较 -->
angular.isArray -------------------------------------------- 是否是数组
angular.isDate --------------------------------------------- 是否是日期
angular.isElement ------------------------------------------ 是否是元素
angular.isFunction ----------------------------------------- 是否是函数
angular.isNumber ------------------------------------------- 是否是数字
angular.isObject ------------------------------------------- 是否是对象
angular.isString ------------------------------------------- 是否是字符串
angular.isUndefined ---------------------------------------- 是否是未定义


<!-- JSON -->
angular.fromJson ------------------------------------------- 反序列化JSON字符串
angular.toJson --------------------------------------------- 序列化JSON字符串
```
